# Teslo API

1. Clonar el repositorio
2. ``` npm i ```

3. Clonar el archivo ```.env.template``` y renombrar a ```.env```
4. Agregar 

4. Levantar la base de datos
```
docker-compose up -d
```

5. Iniciar servidor
```
npm run start:dev
```
